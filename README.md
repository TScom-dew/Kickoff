# 🌿 Hazaribagh Town Guide Website

A modern and responsive city guide website for Hazaribagh, Jharkhand.  
This project showcases local attractions, wildlife, dams, and city information in a clean and professional layout.

---

## 🚀 Live Website

👉 https://hazaribagh-guide.vercel.app/

---

## 📌 Features

- 🌙 Dark Mode Toggle (Saved in Local Storage)
- 📱 Fully Responsive Design
- 🖼 Image Gallery 
- 🗺 Google Maps Embed
- 📩 Contact / Feedback Form
- 📖 FAQ Section
- ⚡ Fast and Lightweight Design

---

## 🛠 Technologies Used

- HTML5
- CSS3 (Grid + Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Vercel Deployment
